# Feed Short Title Generator

Generate concise `short_title` values optimized for mobile and smaller ad placements in Google Shopping.

## Why this matters

The `[short_title]` attribute is recommended and medium-impact in Google Shopping. It provides:
- A shorter title for Display, YouTube, and smaller placements where full titles get truncated
- Control over what shoppers see when space is limited
- Better performance on mobile where only ~50-65 characters display
- Cleaner appearance in Performance Max campaigns

## Google's specifications

- **Max length**: 150 characters (but aim for 65 characters or fewer)
- **Purpose**: Represents the base product, not a specific variant
- **Do NOT include**: variant info (color, size), promotional text, ALL CAPS
- **DO include**: brand, product type, 1-2 key distinguishing features

## Formula

```
Short title = Brand + Product Type + Key Differentiator
```

Examples:
- Full title: "Nike Air Max 90 Essential Heren Sneaker Zwart/Wit Maat 43"
- Short title: "Nike Air Max 90 Essential Sneaker"

- Full title: "Samsung Galaxy S24 Ultra 256GB Smartphone Titanium Blue"  
- Short title: "Samsung Galaxy S24 Ultra 256GB"

- Full title: "Nike Air Max 90 | Sportshop – Heren Sneakers Wit/Zwart (Maat 43)"
- Short title: "Nike Air Max 90 Heren Sneakers"

### Step 2: Short title generation logic

The short title is generated by stripping variant-specific info from the full title and condensing it to the essential product identity.

```python
import re

# Words/patterns to strip from title to create short version
VARIANT_PATTERNS = [
    # Colors (NL, EN, DE, FR, IT, ES, PT — common colors only)
    r'\b(?:zwart|wit|rood|blauw|groen|geel|oranje|paars|roze|bruin|grijs|beige|goud|zilver|'
    r'black|white|red|blue|green|yellow|orange|purple|pink|brown|grey|gray|gold|silver|'
    r'schwarz|weiß|weiss|rot|blau|grün|gelb|rosa|braun|grau|silber|'
    r'noir|blanc|rouge|bleu|vert|jaune|violet|rose|marron|gris|doré|argenté|'
    r'nero|bianco|rosso|blu|verde|giallo|grigio|oro|argento|'
    r'negro|blanco|rojo|azul|amarillo|naranja|morado|marrón|dorado|plata|'
    r'preto|branco|vermelho|castanho|cinzento|dourado|prateado|'
    r'turquoise|bordeaux|navy|mint|khaki|taupe|antraciet|cognac|ecru|indigo|'
    r'türkis|anthrazit|bordeaux|turchese|antracita)\b',
    # Size indicators
    r'\b(?:maat|size|mt\.?)\s*\w+',
    r'\b(?:xxs|xs|s|m|l|xl|xxl|xxxl|2xl|3xl|4xl|5xl)\b',
    # Pipe separators and everything after (often shop name)
    r'\s*\|[^|]*$',
    # Content in parentheses at end (often variant info or shop name)
    r'\s*\([^)]*\)\s*$',
    # "- Buy Now" / "| Shop" type suffixes (all languages)
    r'\s*[-–—]\s*(?:Shop|Webshop|Online|Kopen|Bestellen|Buy now|Order|Free shipping|Official|'
    r'Kaufen|Jetzt kaufen|Kostenloser Versand|Acheter|Commander|Livraison gratuite|'
    r'Acquista|Compra|Spedizione gratuita|Comprar|Envío gratis|Frete grátis).*$',
    # Slash-separated colors: "Zwart/Wit"
    r'\b\w+/\w+\b',
]

# Words to keep (product identity)
KEEP_PATTERNS = {
    'brand', 'product_name', 'product_type', 'model', 'key_feature'
}

def generate_short_title(title, brand='', product_type=''):
    """Generate a short title from the full title.
    Returns (short_title, method)."""

    title_str = str(title).strip() if pd.notna(title) else ''
    if not title_str:
        return '', 'empty'

    short = title_str

    # Step 1: Remove common suffixes (shop name, CTA's — all languages)
    # Pipe separator (universal)
    short = re.sub(r'\s*\|\s*(?:Shop|Webshop|Winkel|Store|Official|Online|Boutique|Negozio|Tienda|Loja).*$', '', short, flags=re.I)
    # NL
    short = re.sub(r'\s*[-–—]\s*(?:Kopen|Webshop|Online kopen|Bestellen|Nu bestellen).*$', '', short, flags=re.I)
    # EN
    short = re.sub(r'\s*[-–—]\s*(?:Buy now|Shop now|Order now|Buy online|Free shipping|Official store).*$', '', short, flags=re.I)
    # DE
    short = re.sub(r'\s*[-–—]\s*(?:Jetzt kaufen|Jetzt bestellen|Online kaufen|Kostenloser Versand|Offizieller Shop).*$', '', short, flags=re.I)
    # FR
    short = re.sub(r'\s*[-–—]\s*(?:Acheter maintenant|Commander|Acheter en ligne|Livraison gratuite|Boutique officielle).*$', '', short, flags=re.I)
    # IT
    short = re.sub(r'\s*[-–—]\s*(?:Acquista ora|Compra ora|Acquista online|Spedizione gratuita|Negozio ufficiale).*$', '', short, flags=re.I)
    # ES
    short = re.sub(r'\s*[-–—]\s*(?:Compra ahora|Comprar online|Envío gratis|Tienda oficial).*$', '', short, flags=re.I)
    # PT
    short = re.sub(r'\s*[-–—]\s*(?:Compre agora|Comprar online|Frete grátis|Loja oficial).*$', '', short, flags=re.I)

    # Step 2: Remove trailing parentheses (often variant info)
    short = re.sub(r'\s*\([^)]*\)\s*$', '', short)

    # Step 3: Remove color words (these are variant-specific)
    for pattern in VARIANT_PATTERNS[:1]:  # Only color patterns
        short = re.sub(pattern, '', short, flags=re.I)

    # Step 4: Remove size indicators (all languages)
    short = re.sub(r'\b(?:maat|size|mt\.?|größe|taille|taglia|talla|tamanho)\s*\w+', '', short, flags=re.I)
    short = re.sub(r'\b(?:xxs|xs|xl|xxl|xxxl|2xl|3xl|4xl|5xl)\b', '', short, flags=re.I)
    # Don't remove standalone S, M, L as they could be model names

    # Step 5: Clean up whitespace and trailing separators
    short = re.sub(r'\s+', ' ', short)
    short = short.strip(' -–—|,')

    # Step 6: Ensure it's under 65 characters
    if len(short) > 65:
        # Try to cut at a natural word boundary
        words = short.split()
        short = ''
        for word in words:
            if len(short) + len(word) + 1 <= 65:
                short = f"{short} {word}".strip()
            else:
                break

    # Step 7: If stripping made it too short or empty, use cleaned original
    if len(short) < 10:
        short = title_str[:65].rsplit(' ', 1)[0] if len(title_str) > 65 else title_str

    return short.strip(), 'generated'
```

### Step 3: Apply and build supplemental feed

```python
id_col = next((c for c in df.columns if c in ['id', 'unique merchant sku', 'merchant item id']), None)
title_col = next((c for c in df.columns if c in ['title', 'titel', 'product name']), None)
brand_col = next((c for c in df.columns if c in ['brand', 'merk', 'manufacturer']), None)
ptype_col = next((c for c in df.columns if c in ['product_type', 'producttype']), None)
st_col = next((c for c in df.columns if c in ['short_title', 'korte titel']), None)

results = {'generated': 0, 'already_had': 0, 'skipped': 0}
short_titles = []

for idx, row in df.iterrows():
    if st_col:
        current = str(row[st_col]).strip() if pd.notna(row[st_col]) else ''
        if current and current.lower() not in ['', 'nan', 'none']:
            results['already_had'] += 1
            short_titles.append(current)
            continue

    short, method = generate_short_title(
        row.get(title_col, ''),
        row.get(brand_col, ''),
        row.get(ptype_col, ''))

    short_titles.append(short)
    if short:
        results['generated'] += 1
    else:
        results['skipped'] += 1

supplemental = pd.DataFrame({
    'id': df[id_col],
    'short_title': short_titles,
})

output_path = "/mnt/user-data/outputs/supplemental_feed_short_title.xlsx"
supplemental.to_excel(output_path, index=False)
```

## Important guardrails

- Never overwrite existing short_title values
- Short title should be ≤65 characters ideally, max 150
- Don't include variant-specific info (color, size) — the short title represents the base product
- Don't include promotional text, ALL CAPS, or special characters for attention
- The short title must still accurately describe the product
- Keep the brand in the short title — it's a key trust signal
- If the original title is already ≤65 characters and clean, just use it as-is
- Language must match the feed language (Dutch feed → Dutch short title)

## Output format

The output is ALWAYS a supplemental feed — never a full copy of the original feed.

- **Single attribute**: 2 columns → `id` + `short_title`
- **Multiple attributes**: 1 `id` column + 1 column per attribute
- Column names are always the **English Google Merchant Center attribute names**
- Format: **Excel (.xlsx)** — user can upload to Google Drive where it automatically converts to a Google Sheet
- Include ALL products (also empty ones)
